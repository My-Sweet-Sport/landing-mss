# Mirage Studio (SvelteKit)

A SvelteKit + runes recreation of the popular shadcn/ui landing page design, implemented with Tailwind CSS and shadcn-svelte-style components.

## Getting started

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

## Tech

- SvelteKit with runes enabled
- Tailwind CSS with shadcn-svelte inspired primitives (button, badge, card, input)
- Lucide icons
